# 🛡️ PROJECT OVERVIEW & GOALS
CyberGuard AI is a multi-vector threat detection and real-time prevention platform designed for enterprise threat monitoring. It ingests URLs, Emails, Network logs, and Web application requests, classifies them using trained machine learning classifiers, enriches them with threat intelligence, and correlates attacks to dynamically adjust risk scores. It includes an active Intrusion Prevention System (IPS) that blocks malicious entities at the middleware layer (403 Forbidden) and runs automated cleanups. It also tracks user activities using User Behavior Analytics (UBA) to detect anomalies (like Impossible Travel or API spikes).

---

# 🏗️ SYSTEM ARCHITECTURE & COMPONENTS

1. FRONTEND: React (Vite) styled with Vanilla CSS Glassmorphism. Features dynamic dashboards, scanner sections, user behavior analytics charts (recharts), threat timelines, alert panels, and a threat prevention center (active block lists, unblock history).
2. BACKEND API: FastAPI (Python 3.11+) + SQLAlchemy ORM + SQLite database. Integrated with JWT auth and multi-tenant workspace isolation.
3. MIDDLEWARE LAYER:
   - AuditMiddleware: Automatically intercepts mutations (POST, DELETE) and logs administrative events for SOC-2 audit compliance.
   - PreventionMiddleware: Extracts client IP, queries active blocklists, and blocks requests with 403 Forbidden if blocked.
4. AI AGENT ORCHESTRATOR (SecurityAgent): Coordinates vector scans using Python's asyncio to process classifications in parallel. Normalizes outputs and calculates unified threat risk scores (0-100).
5. MACHINE LEARNING DETECTORS:
   - URL Scanner: Random Forest model extracting lexical features to flag phishing URLs.
   - Email Scanner: NLP Naive Bayes + TF-IDF vectorizer classifying phishing emails.
   - Network IDS: Random Forest classifier built on NSL-KDD dataset to flag network traffic anomalies.
   - Web Attack Scanner: Custom classifier detecting SQLi and XSS payloads.
6. THREAT INTELLIGENCE: TTL-cached lookups matching domain and IP indicators against cached threat blacklists.
7. CORRELATION ENGINE: Evaluates occurrences of the same indicators across multiple vectors within a 24-hour window, applying scoring multipliers.
8. INTENSE ALERT SYSTEM: Escalates severity (LOW, MEDIUM, HIGH, CRITICAL) when intelligence hits or repeated patterns occur, employing a 60-minute duplicate cooldown. Connects to webhooks and SMTP email templates.
9. INTRA-SYSTEM SCHEDULER: Background cron scheduler running cleanup loops every 5 minutes to release expired blocked IPs.
10. USER BEHAVIOR ANALYTICS (UBA): Builds a 30-day baseline profile of IPs, login hours, and locations per user. Scores events and alerts on anomalies (Impossible travel speed > 900 km/h, credential brute-forcing, API access spikes).
11. EXPLAINABILITY & MITRE ATT&CK: Static mapping pipeline linking classifications to MITRE ATT&CK technique IDs (T1566 Phishing, T1190 Web Exploits, etc.) and generating text explanations with recommended analyst actions.
12. EXECUTIVE PDF GENERATOR: Custom dependency-free PDF layout builder generating assessment reports.
